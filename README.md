A straightforward nutrition reference for Indian fast food. No login, no app, no nonsense just the macro data from official brand websites, all in one place.
Most chains publish their nutrition info somewhere buried in a PDF, tucked away on a menu page, or hidden in an app. This site pulls that data together so you don't have to go hunting every time you want to know what's in your order.

What's inside
15 chains, 175+ items. McDonald's, KFC, Burger King, Domino's, Pizza Hut, Subway, Starbucks, Taco Bell, Wow! Momo, Haldiram's, Chaayos, Faasos, Social, and California Burrito. Each item shows calories, protein, carbs, and fat per serving.
How to use it
Search by item name or chain. Filter by veg or non-veg. Narrow down by category just pizzas, just chai, just burgers. Sort any column up or down to find the highest-protein option on the menu or the lowest-calorie item across all chains.

A few things to keep in mind
All values come directly from official brand websites and published menus. Numbers are per serving as stated by each brand serving sizes vary, so a "per slice" and a "per burger" aren't directly comparable without context. Values are approximate and can vary by outlet, city, and how the item is prepared on a given day.
This site doesn't update in real time if a brand changes their menu, there will be a lag before it's reflected here.
